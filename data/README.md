# Data 模块

## 负责什么
- 负责保存静态数据、规则数据、场景数据、模板数据
- 负责作为底座可读取的数据源
- 负责承载非代码型配置内容

## 不负责什么
- 不负责动态流程执行
- 不负责数据库持久化
- 不负责分析引擎推理

## 输入 / 输出
- **输入**: 规则文件、静态配置、模板文件、场景定义
- **输出**: 可读取的数据对象、规则对象、模板对象

## 目录说明
- `scenarios/` - 场景定义文件（按项目组织）
- `standards/` - 评分标准和规则文件
- `raw/` - 原始数据导入目录
- `scenarios.json` - 场景索引
- `standards.json` - 标准索引
