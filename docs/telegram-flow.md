# Telegram Flow

## Current flow
1. User sends `/start` or `开始`
2. User sends `/score` / `继续` / `再来一题`
3. Bot asks for scenario number
4. User sends scenario number
5. Bot asks for customer message or `默认`
6. User sends customer message or `默认`
7. Bot asks for agent reply
8. User sends reply
9. Bot returns structured training feedback
