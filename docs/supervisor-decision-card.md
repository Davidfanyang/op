# 主管最小判读模板

> 不需要读长文档,快速做判断。打印出来贴桌上也行。

## 一、Training 低分怎么判 (30秒决策卡)

### 看分数 → 看标签 → 做决定

| 分数段 | 问题标签 | 处理决定 | 后续动作 |
|--------|---------|---------|---------|
| **<30分** | 态度问题 | `needs_training` | 立即辅导,跟踪改进 |
| **<30分** | 话术问题 | `needs_training` | 安排专项训练 |
| **<30分** | 流程问题 | `needs_review` | 组织复盘,优化SOP |
| **30-50分** | 任何标签 | `needs_training` | 安排改进训练 |
| **50-70分** | 话术/轻微 | `passed` | 标记误判/可接受 |
| **50-70分** | 流程问题 | `viewed` | 记录备案 |

### 快速判断流程

```
分数<30?
  ├─ 是 → 态度问题? → needs_training (立即辅导)
  │        ├─ 话术问题? → needs_training (专项训练)
  │        └─ 流程问题? → needs_review (复盘SOP)
  │
  └─ 否 → 分数30-50? → needs_training (改进训练)
             └─ 分数>50? → passed (可放过)
```

### 常见场景速查

| 客服回复 | 问题类型 | 处理决定 |
|---------|---------|---------|
| "不知道" | 态度问题 | needs_training |
| "什么问题?" | 话术问题 | needs_training |
| "你等几天" | 流程问题 | needs_review |
| "您好,请问..." | 无问题 | passed |

## 二、Live Monitor 告警怎么判 (1分钟决策卡)

### 看告警级别 → 看路由 → 做决定

| 告警级别 | 路由结果 | 处理决定 | 响应时效 |
|---------|---------|---------|---------|
| **critical** | supervisor_review | `approved` | 立即处理 |
| **warning** | supervisor_review | `approved` / `rejected` | 当天处理 |
| **warning** | observation_queue | 定期review | 本周处理 |
| **observation** | auto_record | 不处理 | 仅统计 |

### 直接进主管 Review 的4种情况

✅ **必须立即处理**:
1. 分数 < 20
2. 态度维度 < 15% (3/20分)
3. 包含风险关键词 ("不知道"、"不关我事"、"没办法")
4. 可能引发投诉/升级

### 只进待观察队列的3种情况

⏸️ **定期review即可**:
1. 分数 20-40 分
2. 某维度较弱 (但不至critical)
3. 员工首次出现问题

### 不进主管流的3种情况

📊 **仅记录统计**:
1. 分数 40-50 分 (observation级)
2. 轻微话术问题
3. 重复低优先级问题 (已触发限流)

## 三、哪些不纳入员工考核

### ❌ 不考核的5种情况

| 情况 | 判断标准 | 处理决定 |
|------|---------|---------|
| **AI误判** | 回复实际没问题,但AI评分低 | `passed` / `rejected` |
| **特殊情况** | 客户情绪激动/系统故障/紧急处理 | `passed` |
| **知识库缺失** | FAQ没有覆盖这个场景 | `needs_review` |
| **流程缺陷** | SOP本身有问题或不清晰 | `needs_review` |
| **系统问题** | 系统限制导致无法更好服务 | `passed` |

### 核心判断原则

```
问题是否在员工可控范围内?
  ├─ 是 → 可以考核 (话术、态度、流程执行)
  └─ 否 → 不考核 (系统、流程、知识库、特殊情况)
```

### 快速检查清单

处理前问自己3个问题:
- [ ] 客服是不是真的有问题? (排除AI误判)
- [ ] 这个问题客服能不能控制? (排除系统/流程问题)
- [ ] 有没有特殊原因? (排除特殊情况)

**3个都是"是" → 纳入考核**  
**任一"否" → 不纳入考核**

## 六、常见问题 (3句话解答)

### Q: 分数多少算低?
**A**: <30严重, 30-50需改进, >50可接受。

### Q: 话术和态度怎么区分?
**A**: 想做好但不会表达=话术; 敷衍不负责=态度。

### Q: 误报率高怎么办?
**A**: 标记passed,记录原因,每周分析优化规则。

### Q: 什么时候用passed,什么时候用rejected?
**A**: training用passed; live_monitor用rejected。

### Q: 怎么判断是不是流程问题?
**A**: 客服按SOP操作但SOP有问题→流程问题,不考核员工。
