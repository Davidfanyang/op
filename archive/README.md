# Archive 目录 - 归档文件

## 目录定位
本目录存放**历史版本代码**和**废弃模块**，用于：
1. 保留历史实现供参考
2. 避免直接删除导致代码丢失
3. 明确标识这些代码不再使用

## 当前归档文件
- `evaluator-legacy-v2.0.js` - 旧版评估器（v2.0）
- `feedback-legacy-v2.0.js` - 旧版反馈构建器（v2.0）

## 使用注意
1. 本目录代码**不参与生产运行**
2. 新版本开发不应依赖本目录代码
3. 确认无参考价值后可安全删除

## 归档原则
当满足以下条件时，代码应移入 archive/：
- 已被新实现完全替代
- 生产环境不再使用
- 可能仍有参考价值的历史代码
